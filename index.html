Smart mail transfer Protocal using Python

This Python script implements a simple OTP-based email verification system using the SMTP protocol. It generates a random 4-digit OTP and sends it to the user's email address via Gmail. The user is then prompted to enter the received OTP for verification. If the entered OTP matches, access is granted; otherwise, the login fails. The script uses Python's smtplib and email modules for sending emails securely with Gmail's SMTP server.
